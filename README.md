# SipaaKernel
What's behind your favorite open-source OS

## Features
* Architecture :
  * x86_64 :
      * Basic IDT
      * PIC
      * GDT & TSS
  * Input / Output
* Devices :
  * Serial
* Libraries
  * Standard Library (libc)
  * Logging library (liblogging)
  * Console library (libconsole)
  * String Management Library (libstr) (empty)
* Graphics
   * Single Buffering
   * Double Buffering
