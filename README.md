# SipaaOS
SipaaOS, with a new homemade base.

## Features
* Architecture :
  * x86_64 :
      * Basic IDT
      * PIC (not used currently)
      * GDT
  * Input / Output
* Devices :
  * PS2 :
    * Mouse (not working)
  * Serial
* Graphics
* Memory Manager (not sure if it's working)
* Standard Library (broken)
* System library (small, but working)
